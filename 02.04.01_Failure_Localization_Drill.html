Failure Localization — vague to precise

Read the vague statement. Room produces the precise version. Then reveal.

Precision test: "Is this specific enough that an engineer could build the missing component from this description alone?" If no — the trainee returns to Step 3.
Kill Switch: trainee states a category-level failure and advances to Step 4 without the specific broken mechanism. Stop. "You have named the category, not the failure. What exactly is absent from the system?"
